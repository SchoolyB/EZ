# EZ Elavator Pitch

### Tagline:
EZ: Programming Made EZ
---
### Brief Description:

EZ is a statically-typed, procedural programming language built to be simpler than Python but closer in spirit to Go and Rust. It rejects classes, hidden behavior, and magic in favor of explicit, beginner-friendly syntax, predictable semantics, and a clean standard library. EZ teaches real fundamentals without the overhead or noise of larger languages.

---
### Long Paragraph Description:

EZ is a modern, interpreted programming language created with a simple belief: learning to program shouldn’t require learning unnecessary complexity.
Where languages like Python hide behavior behind dynamic typing and implicit conversions, EZ stays honest. It uses strong static typing, clear procedural structure, and explicit semantics to teach real programming fundamentals—the same concepts professional developers use in languages like Go, Rust, and C.

EZ deliberately rejects object-oriented programming, package managers, and hard-to-trace abstractions. Instead, it offers a straightforward mix of procedural and functional patterns, Rust-inspired error messages, and a standard library designed around clarity and simplicity. There's one obvious way to write things, but advanced users still have room for control through explicit aliases, safe variants, and predictable behavior.

Whether you're typing your first line of code into the REPL or structuring a larger program with modules, structs, enums, arrays, maps, and sets, the experience is the same: no magic, no guesswork, no surprises. EZ gives beginners a gentle on-ramp while offering experienced developers a refreshing, no-BS language that respects their understanding.

It’s a language with a low barrier to entry and a high skill ceiling—proving that simple doesn’t mean limited, and honest design beats clever complexity every time.
